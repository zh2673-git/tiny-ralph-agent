# Brainstorming Techniques Reference

## 1. SCAMPER
- **S**ubstitute - 替代什么？
- **C**ombine - 组合什么？
- **A**dapt - 如何适应？
- **M**odify - 如何改进？
- **P**ut to other uses - 还有什么其他用途？
- **E**liminate - 可以消除什么？
- **R**everse/Rearrange - 如何反转/重排？

## 2. 6顶思考帽
- 🔴 红色 - 情感和感觉
- 🟡 黄色 - 乐观和价值
- 🟢 绿色 - 创意和新想法
- 🔵 蓝色 - 控制和流程
- ⚪ 白色 - 信息和数据
- ⚫ 黑色 - 风险和问题

## 3.奥斯本检核表
1. 能否他用？现有事物有无新用途？
2. 能否借用？能否借鉴其他经验？
3. 能否改变？颜色、形状、味道等
4. 能否扩大？增加时间、频率、尺寸
5. 能否缩小？压缩、简化的可能
6. 能否替代？材料、方法的替代
7. 能否调整？改变顺序、节奏
8. 能否颠倒？正反、上下互换
9. 能否组合？混合、合成
